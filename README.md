## Bonus 3 — Lighthouse Audit (manual step)

1. Open the page in Chrome, right-click → Inspect → tab **Lighthouse**.
2. Select **Performance** and **Accessibility**, click **Analyze page load**.
3. Screenshot the score ("before").
4. Common quick fixes if score is low:
   - Add explicit `width`/`height` to any images if you add them later.
   - Make sure all text has enough color contrast (check dark mode too).
   - Avoid render-blocking resources; font `display=swap` is already set.
5. Re-run Lighthouse, screenshot the score again ("after").
6. Paste both screenshots here in the README.
